# pid-parse

`pid-parse` is a Rust parser for SmartPlant / Smart P&ID `.pid` files.

## Roadmap

- Parse OLE/CFBF container structure
- Extract `TaggedTxtData` XML metadata
- Index `JSite*` storages and property streams
- Scan cluster and dynamic-attribute streams
- Build a reverse-engineering friendly inspection report

## Status

Initial bootstrap commit. Source skeleton will be added in follow-up commits.
