## CLI 入口索引

```
pid_inspect <file.pid>                        # 默认 human 报告（含 Coverage section）
pid_inspect <file.pid> --coverage             # 仅 Coverage 段 (Phase 10a)
pid_inspect <file.pid> --coverage --json      # Coverage JSON (Phase 10e)
pid_inspect <file.pid> --json                 # 整个 PidDocument JSON
pid_inspect <file.pid> --schema               # JSON Schema 导出
pid_inspect <file.pid> --probe-*              # 逆向辅助 probes
pid_inspect <file.pid> --round-trip <out> [--verify]
pid_inspect <file.pid> --diff <other.pid>
pid_inspect <file.pid> --set-drawing-number NEW --output <out>
pid_inspect <file.pid> --set-xml-tag <stream> <tag> <value> --output <out>

pid_writer_validate <file.pid> [--out <out>] [--keep] [--json] [--quiet]
        [--edit ATTR=VALUE]+                  # Drawing XML 单项 (Phase 9a)
        [--general-edit ELEMENT=VALUE]+       # General XML 单项 (Phase 9b)
        [--set-summary KEY=VALUE]+            # Summary prop 单项 (Phase 9m)
        [--delete-summary KEY]+               # Summary prop 删除 (Phase 9n)
        [--apply-plan <plan.json>]            # Declarative plan (Phase 9k)
```

## Consumer 快速上手

### Parse（v0.5.3+）

```rust
use pid_parse::PidPackage;
let pkg = PidPackage::from_path("input.pid")?;
// 或 from HTTP body:
// let pkg = PidPackage::from_bytes(&body_bytes)?;
```

### Edit via declarative plan

```rust
use pid_parse::{PidPackage, PidWriter, WritePlan};

let pkg = PidPackage::from_path("input.pid")?;
let plan = WritePlan::from_json(r#"{
  "metadata_updates": {
    "drawing_xml": "<Drawing><DrawingNumber>X-9999</DrawingNumber></Drawing>",
    "summary_updates": { "title": "Q4 Pipeline Review", "author": "Jane Engineer" },
    "summary_deletions": ["comments"]
  }
}"#)?;

// To file
PidWriter::write_to(&pkg, &plan, Path::new("output.pid"))?;
// Or bytes (v0.5.3+)
let bytes = PidWriter::write_to_bytes(&pkg, &plan)?;
```

### Inspect coverage programmatically

```rust
use pid_parse::{PidPackage, inspect::coverage};

let pkg = PidPackage::from_path("input.pid")?;
let report = coverage::coverage_report(&pkg.parsed);
for entry in &report.entries {
    if matches!(entry.status, pid_parse::ParseCoverageStatus::Unknown) {
        println!("still unknown: {} ({} bytes)", entry.name, entry.stream_size.unwrap_or(0));
    }
}
// Or JSON for CI (v0.6.4+)
let json = report.to_json_pretty()?;
```
